h2. oui_instagram_author_info

Displays each author info in a @oui_instagram_author@ container tag.

bc. <txp:oui_instagram_author_info />

h3. attributes

- @class@ :=
The css class to apply to the HTML tag assigned to @wraptag@.
*Default:* none
*Valid:* string =:
- @link@ :=
To link the author info to its Instagram page or to his website.
*Default:* none
*Valid:* @instagram@, @website@ =:
- @type@ :=
The information type to display.
*Default:* @caption@
*Valid:* @id@, @username@, @full_name@, @avatar@, @bio@, @followers_count@, @following_count@, @medias_count@ =:
- @wraptag@ :=
The HTML tag to use around the generated content.
*Default:* none
*Valid:* HTML container tag without brakets =:
