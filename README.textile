h2. Basics

Easily display recent images from an Instagram account.

notextile. <hr id="requirements" />

h2. Plugin requirements

oui_instagram’s minimum requirements:

* Textpattern 4.6+

notextile. <hr id="install" />

h2. Installation

# "Download":https://github.com/NicolasGraph/oui_instagram/releases the plugin;
# Paste the content of the plugin file under the *Admin > Plugins*, upload it and install;
# Click _Options_ or visit your *Admin>Preferences* tab to fill the plugin prefs.

notextile. <hr id="prefs" />

h2. Preferences / options

* *Access token* - _Default: unset_ - A valid Instagram access token.

p(alert-block warning). %(ui-icon ui-icon-alert).% you can easily get an access token from Instagram or by using a webservice like "Pixel Union":http://instagram.pixelunion.net/. However by default, this access token will allow you to display images from your own account only. If you need to use other accounts, you will need to create an Instagram client and invite users in the Sandbox, or ask a special permission by registering your app. See the "Instagram developer documentation":https://www.instagram.com/developer/ for more informations.

p(alert-block information). %(ui-icon ui-icon-info).% From v0.6.9, you can also use the @access_token@ attribute as an alternative to provide several Instagram access token.

* *Default cache time* — _Default: 0_ - Duration of the cache in minutes.

h2. Tags

h3(#oui_instagram_images). oui_instagram_images

Displays a recent images list.
Can be used as a single or a container tag.

bc. <txp:oui_instagram_images />

or

bc. <txp:oui_instagram_images>
[…]
</txp:oui_instagram_images>

- @access_token@ :=
the access token provided via the plugin prefs.
*Default:* none
*Valid:* a different Instagram access token than the one provided via the plugin prefs =:
- @break@ :=
he HTML tag used around each generated image
*Default:* @li@
*Valid:* HTML self-closing tag without brakets =:
- @cache_time@ :=
Duration of the cache in minutes
*Default:* @5@
*Valid:* integer =:
- @class@ :=
The css class to apply to the HTML tag assigned to @wraptag@.
*Default:* @oui_instagram_images@
*Valid:* string =:
- @label@ :=
The label used to entitled the generated content.
*Default:* none
*Valid:* string =:
- @labeltag@ :=
The HTML tag used around the value assigned to @label@.
*Default:* none
*Valid:* HTML container tag without brakets =:
- @limit@ :=
The number of images to display.
If the @limit@ value is greater than _20_, several requests will be thrown to pull your images
*Default:* @10@
*Valid:* integer =:
- @link@ :=
To apply a link around each generated image to the standard_resolution image.
*Default:* @auto@
*Valid:* @auto@ (linked to the Instagram page), @1@ (linked to the image url), @0@ =:
- @type@ :=
The image type to display.
Should be set in the @<txp:oui_instagram_image />@ tag if @<txp:oui_instagram_images />@ is used as a container
*Default:* @thumbnail@
*Valid:* @thumbnail@, @low_resolution@, @standard_resolution@ =:
- @user_id@ :=
Provide an Instagram username to generate a gallery from different account than the one associated with the access token plugin preference; it is faster than username
*Default:* none
*Valid:* string =:
- @username@ :=
Provide an Instagram username to generate a gallery from different account than the one associated with the access token plugin preference
*Default:* none
*Valid:* string =:
- @wraptag@ :=
The HTML tag to use around the generated content.
*Default:* @ul@
*Valid:* HTML container tag without brakets =:
